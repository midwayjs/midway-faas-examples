# Midway Serverless

## 快速开始

> 国内用户建议使用 `cnpm` 加速npm，`npm install -g cnpm --registry=https://registry.npm.taobao.org`


### 第一步：安装 serverless

```shell script
npm i -g serverless
```

### 第二步：创建项目

```shell script
$ serverless install --url https://github.com/midwayjs/midway-faas-examples/tree/master/demo-faas
```

### 第三步：进入目录

```shell script
$ cd demo-faas
```

### 第四步：安装npm依赖

```shell script
$ npm i
```
